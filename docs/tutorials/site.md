# Bibliography Site Walkthrough

`biblio site` turns a bibliography workspace into a local static portal.

## Build the site

```bash
biblio site build
```

## Serve it locally

```bash
biblio site serve
```

## What gets generated

By default, `biblio` writes to `bib/site/`.

Important outputs:

- `bib/site/index.html`
- `bib/site/papers/`
- `bib/site/status.html`
- `bib/site/graph.html`
- `bib/site/_data/papers.json`
- `bib/site/_data/artifacts.json`
- `bib/site/_data/status.json`
- `bib/site/_data/graph.json`
