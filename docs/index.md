# biblio

`biblio` is a portable bibliography workspace tool.

It helps a project create and maintain a repo-local `bib/` workspace for:

- BibTeX source management
- PDF fetch and normalization
- Docling-derived markdown and sidecars
- OpenAlex metadata enrichment
- bibliography-owned RAG source config
- static bibliography site generation

The docs follow the Diataxis structure:

- Tutorials: end-to-end guided paths
- How-to guides: task-focused recipes
- Explanation: design choices and concepts
- Reference: command and file layout details
