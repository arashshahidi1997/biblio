# Roadmap

The system is designed to evolve incrementally without architectural rewrites.

## Near-term

- Stable Bronze/Silver/Gold pipeline
- Citation-context indexing
- Project-scale RAG with grounding guarantees

## Mid-term

- Citation intent classification
- Claim–evidence linking
- Figure- and table-aware retrieval

## Long-term

- Manuscript-aware assistance
- Cross-project knowledge synthesis
- Public-facing bibliographic intelligence services

Throughout, the core invariants remain:
- reproducibility,
- traceability,
- and explicit human-in-the-loop correction.

This is not a tool that replaces researchers — it is infrastructure that **amplifies scientific reasoning**.
