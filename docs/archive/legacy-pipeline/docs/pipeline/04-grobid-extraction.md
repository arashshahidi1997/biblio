# GROBID: Bibliographic Intelligence

GROBID provides **bibliographic semantics** at the PDF level, externalizing how papers reference one another into an inspectable record.

## Responsibilities

- Extract title, authors, affiliations.
- Parse the reference list into structured records.
- Align in-text citation markers to references.

## Key capability

GROBID answers:
> “Where in this document is this paper cited?”

This capability is essential for citation-aware reasoning.

## Outputs

- Parsed references with metadata.
- In-text citation locations.
- Mapping between citation markers and references.

GROBID does not understand document layout;
it complements Docling rather than competing with it.
