# Project Bibliography Intelligence

This project develops a **project-scoped bibliography intelligence system** for scientific research—an infrastructure designed to make scientific reasoning more transparent, grounded, and scalable.

The goal is to move beyond traditional reference managers toward a system that:
- grounds scientific statements in **exact document locations**,
- understands **how papers cite one another**,
- and supports **retrieval-augmented reasoning** over a curated body of literature as a computable, inspectable mental model of a field.

Rather than treating PDFs, metadata, and citation graphs as separate concerns, the system integrates:
- structured document extraction,
- bibliographic parsing,
- global citation graphs,
- and semantic retrieval.

The result is a bibliography that functions not only as a list of references, but as a **computable knowledge substrate** for analysis, writing, and collaboration—an externalized model grounded in documents and citations that experts can inspect and extend.
