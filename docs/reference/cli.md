# CLI

Main command groups:

- `biblio init`
- `biblio bibtex ...`
- `biblio docling ...`
- `biblio openalex ...`
- `biblio rag ...`
- `biblio graph ...`
- `biblio site ...`

For full command help:

```bash
biblio --help
biblio bibtex --help
biblio docling --help
biblio openalex --help
biblio site --help
```
