# CLI

Main command groups:

- `biblio init`
- `biblio ingest ...`
- `biblio bibtex ...`
- `biblio docling ...`
- `biblio openalex ...`
- `biblio rag ...`
- `biblio graph ...`
- `biblio site ...`
- `biblio ui ...`

For full command help:

```bash
biblio --help
biblio ingest --help
biblio ingest pdfs --help
biblio bibtex --help
biblio docling --help
biblio openalex --help
biblio site --help
biblio ui --help
```
