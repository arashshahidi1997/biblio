# Paper comparison tables

## Overview
- `biblio compare citekey1 citekey2 ...` → structured comparison along key dimensions
- Output as markdown table, directly usable in related work sections

## Dimensions
- Method / approach
- Dataset(s) used
- Evaluation metrics
- Key findings / results
- Limitations
- Computational requirements (if applicable)

## Approach
- Requires paper summaries or at least docling extractions for each paper
- LLM extracts comparable features and aligns them across papers
- Output: markdown table + optional prose synthesis
- Store as `bib/derivatives/comparisons/{citekey1}_vs_{citekey2}.md` or by topic

## Integration
- MCP tool: `biblio_compare(citekeys, dimensions=None)`
- CLI: `biblio compare <citekey1> <citekey2> [...]`
- Could feed into presentation generation and lit reviews
