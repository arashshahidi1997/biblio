# Concept extraction and cross-paper linking

## Overview
- Extract key concepts, methods, and techniques from each paper
- Build a concept graph linking papers through shared concepts
- Enables queries like "which papers use variational inference?" without keyword search

## Approach
- Store as structured metadata alongside summaries: `bib/derivatives/concepts/{citekey}.yml`
- Concept taxonomy: methods, datasets, metrics, domains, techniques
- Cross-paper linking via shared concept nodes
- Could feed into RAG as an additional indexed source

## Use cases
- "Find all papers using method X" — concept-level, not keyword-level
- Visualize concept clusters in the library
- Identify methodology gaps: "I have papers on X and Y but nothing connecting them"
- Inform reading list prioritization
