# Deep research literature review workflow

## Overview
- Multi-step AI workflow analogous to ChatGPT's "deep research" but grounded in the user's own library
- Two modes: seed-expand-synthesize and query-driven

## Seed-expand-synthesize mode
- Start from a few citekeys → `graph_expand` to find related work via OpenAlex
- Ingest top candidates → summarize each → synthesize across all into a review narrative
- Iterative: agent judges relevance, expands further if gaps remain
- Output: structured notio note with sections, inline `\cite{citekey}` references, identified gaps

## Query-driven mode
- "What methods exist for X?" → `rag_query_multi` across indexed papers
- Synthesize findings with citations back to library citekeys
- Lighter weight, no new ingestion required

## Automation
- Inherently multi-step and judgment-heavy — ideal for Claude jobs / worklog tasks
- A `lit-review` notio note type with a template for the output structure
- Worklog prompt template that chains existing MCP tools: graph_expand → ingest → summarize → synthesize
