# Presentation generation from paper context

## Overview
- Generate shareable slide decks from paper context using markdown-based presentation tools
- Leverage docling's extracted figures and structured content via markdown transclusion
- Output to `bib/derivatives/slides/{citekey}.md`

## Approach
- Marp (lightest option): markdown with `---` slide separators, renders to HTML/PDF/PPTX
- Alternatives: Slidev (more interactive), reveal.js (more customizable)
- Extract key figures from docling JSON output
- Pull summary points from the summary derivative (depends on paper summary pipeline)
- `biblio present <citekey>` CLI command

## Image transclusion
- Docling already extracts figures with captions
- Markdown image references can point to extracted figure files
- Marp supports standard markdown image syntax with sizing directives

## Automation
- Single-shot AI: assemble context → LLM generates slide markdown
- Could offer templates: "journal club", "conference talk", "lab meeting"
