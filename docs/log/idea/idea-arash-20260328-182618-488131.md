# Citation paragraph drafting tool

## Overview
- Given a claim or section heading + the user's library, draft a paragraph with proper `\cite{citekey}` references
- The RAG layer already supports this; wrapping it as a dedicated tool makes it frictionless

## Approach
- Input: claim, section heading, or topic sentence
- `rag_query_multi` to find relevant passages across indexed papers
- LLM drafts paragraph grounded in retrieved content with inline citations
- Output uses `\cite{citekey}` or `[@citekey]` (configurable for LaTeX or Pandoc)

## Integration
- MCP tool: `biblio_cite_draft(text, style="latex", max_refs=5)`
- Works best after papers have been docling-processed and RAG-indexed
- Could be a building block for the lit review workflow
