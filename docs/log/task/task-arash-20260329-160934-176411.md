# Feed user notes and annotations into AI derivative context

## Goal

Include user-written notes (`bib/notes/{citekey}.md`) and PDF annotations (`bib/derivatives/annotations/{citekey}.json`) in the context assembled for AI derivatives (summaries, concepts, cite-draft, lit review, presentations). This gives the LLM the researcher's perspective.

## Context

- `src/biblio/summarize.py` has `assemble_context(citekey, root)` that gathers BibTeX + docling + grobid — but NOT user notes or annotations
- `src/biblio/cite_draft.py`, `lit_review.py`, `present.py`, `concepts.py`, `compare.py`, `reading_list.py` each assemble their own context
- User notes at `bib/notes/{citekey}.md` — free-form markdown
- PDF annotations at `bib/derivatives/annotations/{citekey}.json` — highlights with optional notes
- The user's notes and highlights represent what they found important — this should guide summaries toward the user's interests
- Should be optional: `include_notes=True` (default True), so `--prompt-only` shows what gets included
