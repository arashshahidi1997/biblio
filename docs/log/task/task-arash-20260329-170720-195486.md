# Related work section generator for academic writing

## Goal

Generate publication-ready "Related Work" sections with thematic subsections, proper citation density, and positioning relative to the user's own contribution. Goes beyond the existing lit review synthesis (which produces a general overview) to produce structured, submission-ready text.

## Context

- Existing `lit_review.py` does query-driven synthesis — good for exploration, but output is a flat narrative, not a structured Related Work section
- Existing `cite_draft.py` drafts single paragraphs — building block, not a section generator
- A Related Work section needs: thematic grouping, transition sentences between groups, positioning statement ("unlike X, our approach..."), proper citation density (2-5 cites per paragraph)
- Input: the user's contribution description + selected papers from library organized by theme/group
- Should support both LaTeX (`\cite{}`) and Pandoc (`[@]`) citation styles
- The user may want to specify grouping (e.g. "group these 5 papers under 'attention mechanisms'") or let the LLM auto-group from concepts
