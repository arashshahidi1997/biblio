# Presentation generation from paper context

## Goal

Generate shareable slide decks from paper context using Marp (markdown-based), with figure transclusion from docling outputs. Supports multiple presentation templates (journal club, conference talk, lab meeting).

## Context

- Source idea: `idea-arash-20260328-182616-769030`
- Docling already extracts figures with captions from PDFs
- Marp is the lightest option: markdown with `---` separators, renders to HTML/PDF/PPTX
- Depends on task 4 (summaries) for structured content to pull into slides
- Should support `--prompt-only` mode
