# Citation drafting and literature review workflow

## Goal

Implement two writing-assistance features: (1) citation paragraph drafting — given a claim, draft a paragraph with `\cite{citekey}` references grounded in the user's library, and (2) a deep research literature review workflow that chains graph expansion, ingestion, summarization, and synthesis.

## Context

- Source ideas: citation drafting (`idea-arash-20260328-182618-488131`), lit review (`idea-arash-20260328-182616-291114`)
- Citation drafting uses RAG (indexio) to find relevant passages, then LLM to draft
- Lit review has two modes: seed-expand-synthesize (multi-step, ideal for worklog tasks) and query-driven (single-shot, ideal for MCP tool)
- Depends on task 4 (summaries) for synthesis input
- Should define a `lit-review` notio note type for structured output
