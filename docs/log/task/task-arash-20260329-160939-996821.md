# Claim-to-evidence linking and verification tool

## Goal

Given a claim from the user's draft, find papers in the library that support, contradict, or qualify it, with exact passages and page references. Stronger than cite-draft (which generates text) — this is about verification and evidence mapping.

## Context

- `cite_draft.py` generates new paragraphs — this tool verifies existing claims
- RAG search finds relevant passages but doesn't classify them as supporting/contradicting/qualifying
- Docling extracts full text with page structure — can provide page-level references
- The tool should help researchers: "Can I actually cite this claim? What's the evidence?"
- Output should be structured: supporting evidence, contradicting evidence, qualifications, missing evidence
- Should also work for reviewing others' papers: "Does this claim hold up against my library?"
