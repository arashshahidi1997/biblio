# GUI: Concept viewer, comparison, and reading list panels

## Goal

Add concept display to PaperTab sidebar, a multi-paper comparison panel, and a reading list curation tool accessible from the ExploreTab or as a new top-level tab.

## Context

- PaperTab right sidebar currently has: LibraryPanel, Related local papers, Neighborhood
- Concepts are per-paper YAML: methods, datasets, metrics, domains, techniques
- Comparison requires selecting multiple papers — ExploreTab already supports context menus on papers
- Reading list takes a research question and returns ranked unread papers
- Backend endpoints: `GET /api/papers/{citekey}/concepts`, `GET /api/concepts/index`, `POST /api/concepts/search`, `POST /api/actions/compare`, `POST /api/reading-list`
- The app has a context menu system (`contextMenu` state in App.jsx)
