# Controlled tag vocabulary and BibTeX keyword sync

## Goal

Establish a faceted tag namespace convention (`domain:`, `method:`, `task:`, `corpus:`) with a vocabulary file for autocomplete/validation, and implement BibTeX keyword extraction on ingest as the first source of tags.

## Context

- Source ideas: `idea-arash-20260328-182638-621544` (vocabulary) and Tier 1 of `idea-arash-20260328-182639-622041` (keyword sync)
- Tags in `bib/config/library.yml` are currently free-form strings — no structure or validation
- BibTeX entries often have a `keywords` field from publishers — free signal that goes unused
- This is the foundation for auto-tagging (task 3) and smart collections (task 2)
- No AI/LLM dependency — pure deterministic tooling
