# PDF highlighting UI with annotation storage

## Goal

Replace the basic PDF iframe viewer in PaperTab with a pdf.js-based viewer that supports text highlighting, annotations, and note-taking, with annotations persisted to the backend.

## Context

- PaperTab currently shows PDFs via `<iframe src="/api/files/pdf/{citekey}">` — browser's native PDF viewer, no annotation support
- Zotero-style annotation: select text → highlight (color choice) → optionally add a note to the highlight
- Annotations should be stored as JSON in `bib/derivatives/annotations/{citekey}.json` — the same location the PDF annotation extraction task (#12) uses
- `react-pdf-highlighter` is a purpose-built React library for this: text selection → highlight creation → persistence. It uses pdf.js under the hood.
- Annotations should be viewable alongside the PDF (sidebar or overlay)
- Existing PDF annotations imported from other tools (via task #12) should also be displayed
